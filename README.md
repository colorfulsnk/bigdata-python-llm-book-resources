# 大数据分析：Python实践与大型语言模型应用

本仓库用于维护教材《大数据分析：Python实践与大型语言模型应用》的配套课程资源。

## 内容结构

- `code/`：教材涉及的示例代码、数据处理脚本与演示文件
- `exercises/`：按章节或按节整理的本节练习题

## 建议组织方式

### 代码

建议按章节建立子目录，例如：

- `code/ch01/`
- `code/ch02/`
- `code/ch03/`

每章目录下可继续放置：

- 示例脚本
- Notebook
- 样例数据说明

### 练习题

建议按章节或节次建立文件，例如：

- `exercises/ch01.md`
- `exercises/ch02.md`

如果需要更细，也可以按节拆分，例如：

- `exercises/ch01/sec01.md`
- `exercises/ch01/sec02.md`
